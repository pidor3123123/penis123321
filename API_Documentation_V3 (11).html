Lead Registration

POST /api/v3/integration?api_token=API_AFFILIATE_TOKEN

POST Data

Required fields are in bold.

Param Value type / Example Description
link_id integer Link ID
fname test First Name
email [email protected] Email
fullphone +11111111 Phone Format (E.164) with plus sign
ip 8.8.8.8 Lead IP
country string Country of lead Format ISO 3166-1 alpha-2
language string Lead's language Format ISO 639-1
source source Source Name
funnel funnel-name Funnel Name
lname test Last Name
pass string User password
domain test.com Domain where form installed
description string Some lead description (quiz answers, etc.)
b_bid string Tracking Data
c_cid string Tracking Data
utm_source string Tracking data
utm_campaign string Tracking data
utm_medium string Tracking data
utm_term string Tracking data
utm_content string Tracking data
click_id string Using in Postbacks

Success Result:

{
    "success": true,
    "id": LEAD_ID,
    "autologin": AUTOLOGIN_URL,
    "password": LEAD_PASSWORD,
}

Error Result:

{
    "success": false,
    "message": ERROR_DESCRIPTION
}

List of error messages:

  • Limits are exhausted!
  • Link not found!
  • Affiliate not found!
  • Recipient not found!
  • Duplicate!
  • Email not invalid!
  • Phone number not valid!
  • First name not valid!
  • Last name not valid!
  • Error! (Unrecognized error)

PHP: 
$domain = '{DOMAIN_NAME}';
$apiToken = '{AFFILIATE_API_TOKEN}';
$url = $domain . '/api/v3/integration?api_token=' . urlencode($apiToken);

$data = [
    'fname' => 'Tom',
    'lname' => 'Hardy',
    'email' => '[email protected]',
    'fullphone' => '+441268272099',
    'ip' => '81.2.69.142',
    'country' => 'GB',
    'language' => 'en',
    'link_id' => 'provided link id',
    'funnel' => 'test funnel',
    'source' => 'FB',
];

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, ['Content-Type: application/x-www-form-urlencoded']);
curl_setopt($ch, CURLOPT_POSTFIELDS, http_build_query($data));
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
curl_close($ch);

Python: 
import requests

url = "https://{DOMAIN_NAME}/api/v3/integration"
params = {
    "api_token": "{AFFILIATE_API_TOKEN}"
}

data = {
    "fname": "Tom",
    "lname": "Hardy",
    "email": "[email protected]",
    "fullphone": "+441268272099",
    "ip": "81.2.69.142",
    "country": "GB",
    "language": "en",
    "link_id": "provided link id",
    "funnel": "test funnel",
    "source": "FB"
}

response = requests.post(url, params=params, data=data, headers={
    "Content-Type": "application/x-www-form-urlencoded"
})

print("Status code:", response.status_code)
print("Response JSON:", response.json())

Leads List

GET /api/v3/get-leads?api_token=API_AFFILIATE_TOKEN

GET Params

Param Value Description
from Y-m-d H:i:s Default start of 30 days ago
to Y-m-d H:i:s Default now
link_id integer ID of link
order_by DESC or ASC DESC
limit integer Query limit
offset integer Query offset
filters[id] integer Lead ID in system
filters[email] string Lead email
filters[acq] integer 0 - NO | 1 - YES
filters[received_by_recipient] integer 0 - NO | 1 - YES

Success Result:

{
    "data": [
        {
            "id": LEAD_ID,
            "source": SOURCE,
            "status": LEAD_STATUS,
            "link_id": LINK_ID,
            "domain": DOMAIN,
            "acq": 0, // 1 - Yes, 0 - No
            "created_at": REGISTRATION_DATE,
        },
        ...
    ],
    "meta": {
        "count": CURRENT_COUNT,
        "next_page": BOOLEAN (true/false)
    },
    "success": true
}

Error Result:

{
    "success": false,
    "message": ERROR_MESSAGE,
    "status_code": 500
}

Possible errors:

  • Unauthenticated.
  • Invalid date format.
